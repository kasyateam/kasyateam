Primary requirements KASYA Develop

Copyright

All submitted projects must be provided by the authors. We ask you to provide exactly your projects. If possible, prove the authorship. (If the authorship is proven, then your project immediately goes to check)

Malicious software

Each project is checked by KASYA Team employees for malware. If suspicious scripts are found, you will be asked questions. If the presence of malware finally appears, then you will be blacklisted.

No monetization, ads or spam

We prohibit the inclusion of elements for monetization, advertising, etc. in our projects. We strictly refer to spam, if detected, there is a possibility of being blacklisted.

Preparation of contract

After drawing up the contract, we DO NOT claim the authorship of your project. This agreement will spell out the services we provide and mutual consent to include your project in the service.

Requirements for hosting projects with demo on KASYA Apps

Structure

We host projects based on the work of web technologies.

For best results, we recommend game engines like Construct 2/3 (If you don't know a programming language), as well as Unity.

Please note that all posted projects are not checked by the system, but by the employees of the KASYA Team. Before posting, you will have to give the source code (under the contract, we do not change the structure of your application) to check for malware.

Also, you need to attach the "ReadMe.txt" file, in which you need to indicate the name of the project, a small description. Based on this data, we will create a personal game page.

Punishment and blacklist system

KASYA Team

How the ban system works:

Since all the basic information flows through the employees of the KASYA Team, we are monitoring. To identify violators, a database is created (This database does not go beyond the KASYA Team.) When you re-submit an application, we compare you with the data in the database and determine there already. If we have any questions, we will interview you.

Reasons for which you can be blacklisted by KASYA Team:

  1. For attempting to host malware.
    Eternal ban.
  2. Repeated violation of established rules.
    From one month to 3 months.
  3. Flood / spam
    From one month to 3 months
  4. Repeated provocations towards developers KASYA Team.
    From 3 months to a year
  5. Attempts to hack servers.
    Eternal ban

    6. This list will be further supplemented.
